##########################################################################################
##### How To Extend the UW_Madison_Omega theme
##########################################################################################

Place this folder at /sites/all/libraries.
Download the crest.png, favicon.ico, wordmark.gif here. 

When you download a new version of the theme, your custom css won't be deleted if you place it here.

If you change any of the theme's settings, don't revert the theme when you download a new version.  This
would cause all the changes you have made to be deleted.  Instead, export your .info file whenever you
download a new version of the theme.
